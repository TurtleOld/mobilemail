# Быстрый старт

Быстрое руководство по началу работы с MobileMail.

## Первый запуск

1. Запустите приложение
2. Введите адрес вашего почтового сервера
3. Нажмите "Войти"

## OAuth авторизация

### Шаг 1: Ввод адреса сервера

Введите адрес вашего почтового сервера, например:
```
https://mail.pavlovteam.ru
```

### Шаг 2: Получение кода авторизации

Приложение автоматически:

1. Получит метаданные OAuth сервера
2. Запросит device code
3. Покажет вам user code и ссылку для авторизации

### Шаг 3: Авторизация в браузере

1. Нажмите кнопку "Открыть" для перехода в браузер
2. Введите user code на странице авторизации
3. Подтвердите доступ для приложения

### Шаг 4: Автоматическое подключение

Приложение автоматически:

- Опрашивает сервер до получения токена
- Сохраняет токены в безопасном хранилище
- Подключается к почтовому ящику

## Работа с почтой

После успешной авторизации вы сможете:

- Просматривать список писем
- Читать письма
- Отправлять письма
- Управлять папками
